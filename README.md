# 💻 Portfolio – Lucas Tadeu

> Um portfólio totalmente feito **sem frameworks**, mostrando que dominar o básico é mais poderoso do que depender de ferramentas prontas.
> Visite o portfólio: https://portfolio-ten-coral-eidf90dyd0.vercel.app/

---

## 🚀 Sobre

Este é o meu portfólio pessoal, criado do zero apenas com **HTML, CSS e JavaScript puro**. Diferente de muitos portfolios modernos que dependem de frameworks e bibliotecas externas, este projeto mostra o que realmente acontece por trás de uma aplicação web: cada interação, cada efeito e cada funcionalidade é controlada diretamente por código escrito por mim.

O objetivo é simples: provar que **entender o básico profundamente vale mais do que saber usar dezenas de frameworks sem compreensão real**.

## 🎯 Motivação

Nos últimos anos, vemos uma tendência crescente de desenvolvedores se tornarem “escravos de frameworks”: decoram bibliotecas e templates, mas não entendem o que está acontecendo por baixo.  

Este portfólio é um manifesto em código: **o básico bem feito supera qualquer solução complexa mal compreendida**.
